SmartPocket is a voice-based money management application designed for teenagers. It helps users manage expenses, build healthy financial habits, and learn basic financial concepts through simple conversation and interactive features.


**Tech Stack**
Frontend: React, Vite
Backend: Node.js, Express
AI Integration: Gemini API


**Installation & Setup**
Prerequisites
Node.js (v16 or above)

▶️ Running the Application

Run Frontend
```npm install```
```npm run dev```

Run Backend
```cd server```
```node index.js```

**Environment Variables**
Create a .env file inside the server folder and add:
```GEMINI_API_KEY=your_api_key_here```
